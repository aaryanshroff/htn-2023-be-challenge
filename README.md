# Instructions
## Docker
```
docker compose up
```
## venv
### Prerequisites
```
export FLASK_APP=app.py
export API_URL=<json_data_source>
```

Run server
```
python3 -m venv .venv
source .venv/bin/activate
flask run
```

# Frameworks and Languages
- Flask (REST API)
- Flask-Caching
- SQLite (Database)
- Flask-SQLAlchemy (ORM)
- Docker
